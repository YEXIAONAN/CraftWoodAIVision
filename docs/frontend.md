# Frontend Tech Stack

## Vue 3 + Vite
- **Role**: Admin dashboard, inspection workflow, traceability page, data dashboard
- **Why**: Component-based development, fast build speed, mature ecosystem
- Composition API throughout

## TypeScript
- **Role**: Type-safe interface contracts across the frontend
- **Why**: Reduces maintenance cost in large frontend projects

## Element Plus
- **Role**: Form building, tables, dialogs, permission menus
- **Why**: Comprehensive admin UI components for rapid development

## Pinia + Vue Router
- **Role**: State management and routing
- **Why**: Clean state for user auth, permission menus, page transitions

## ECharts
- **Role**: Data dashboards and visualization
- **Why**: Rich charting capabilities — defect distribution, inspection trends, after-sales analysis, scan statistics
